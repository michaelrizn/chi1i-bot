# Анализатор совместимости специй

Интеллектуальное веб-приложение для анализа совместимости специй с продуктами и способами приготовления с научным обоснованием.

## 🌟 Особенности

- **Научный подход**: Анализ основан на химических соединениях специй
- **Telegram Mini App**: Полная интеграция с Telegram WebApp API
- **Адаптивный дизайн**: Оптимизировано для мобильных устройств
- **Офлайн поддержка**: Service Worker для работы без интернета
- **Модульная архитектура**: Чистый код без внешних зависимостей

## 🚀 Быстрый старт

### Локальная разработка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/chi1i-bot.git
cd chi1i-bot
```

2. Запустите локальный сервер:
```bash
# Используйте любой статический сервер
python -m http.server 8000
# или
npx serve .
# или
php -S localhost:8000
```

3. Откройте браузер: `http://localhost:8000`

### Деплой на GitHub Pages

1. **Подготовка репозитория**:
   - Убедитесь, что все файлы находятся в корне репозитория
   - Проверьте, что `index.html` находится в корневой папке

2. **Настройка GitHub Pages**:
   - Перейдите в Settings → Pages
   - Выберите Source: "Deploy from a branch"
   - Выберите Branch: "main" (или "master")
   - Выберите Folder: "/ (root)"
   - Нажмите "Save"

3. **Проверка деплоя**:
   - Приложение будет доступно по адресу: `https://your-username.github.io/chi1i-bot/`
   - Деплой может занять несколько минут

## 📱 Интеграция с Telegram

### Создание Telegram Mini App

1. **Создайте бота**:
   - Напишите [@BotFather](https://t.me/botfather)
   - Используйте команду `/newbot`
   - Следуйте инструкциям для создания бота

2. **Настройте Mini App**:
   - Используйте команду `/newapp`
   - Выберите вашего бота
   - Укажите URL: `https://your-username.github.io/chi1i-bot/`
   - Добавьте описание и иконку

3. **Тестирование**:
   - Откройте бота в Telegram
   - Нажмите на кнопку "Открыть приложение"

## 🧪 Алгоритм анализа

### Расчет совместимости

1. **Базовый балл**: 50 очков
2. **Совместимость с продуктами**: +15 очков за каждое совпадение
3. **Совместимость с методами**: +10 очков за каждое совпадение
4. **Химические синергии**: +5 очков за каждое общее соединение
5. **Конфликты**: -20 очков за каждый конфликт
6. **Температурные конфликты**: -10 очков за каждый конфликт

### Научное обоснование

- Анализ химических соединений
- Температурная совместимость
- Вкусовой баланс
- Полезные свойства
- Рекомендации по улучшению
