# Chi1i Bot - Анализ совместимости специй

Умный помощник для анализа совместимости специй и создания идеальных вкусовых сочетаний.

## 🌶️ Возможности

- **Анализ совместимости** - Детальный анализ совместимости специй с продуктами и методами приготовления
- **Химический анализ** - Анализ химических взаимодействий между специями
- **Вкусовой профиль** - Оценка доминирующих нот, баланса и сложности вкуса
- **Рекомендации** - Персонализированные рекомендации по использованию специй
- **PWA поддержка** - Работает как нативное приложение на мобильных устройствах
- **Telegram WebApp** - Интеграция с Telegram для удобного использования

## 🚀 Технологии

- **React 18** - Современная библиотека для создания пользовательских интерфейсов
- **TypeScript** - Типизированный JavaScript для надежности кода
- **Vite** - Быстрый инструмент сборки
- **Tailwind CSS** - Utility-first CSS фреймворк
- **React Router** - Маршрутизация для SPA
- **Lucide React** - Современные SVG иконки
- **PWA** - Progressive Web App функциональность
- **Service Worker** - Кэширование и офлайн работа

## 📱 Архитектура

### Компоненты
- `Button` - Кнопки с различными вариантами стилей
- `Card` - Карточки для группировки контента
- `Input` - Поля ввода с поддержкой поиска
- `Select` - Выпадающие списки
- `Badge` - Метки и теги
- `Loading` - Индикаторы загрузки
- `Navigation` - Нижняя навигация

### Страницы
- `Home` - Главная страница с каталогом специй
- `Analysis` - Страница анализа совместимости
- `Settings` - Настройки приложения

### Утилиты
- `SpiceCompatibilityAnalyzer` - Система анализа совместимости
- `DataLoader` - Загрузка и кэширование данных
- `useTelegram` - Хук для работы с Telegram WebApp
- `useServiceWorker` - Хук для PWA функциональности

## 🛠️ Установка и запуск

### Предварительные требования
- Node.js 20+
- npm или pnpm

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

### Сборка для продакшена
```bash
npm run build
```

### Предварительный просмотр сборки
```bash
npm run preview
```

### Проверка типов
```bash
npm run check
```

### Деплой на GitHub Pages
```bash
npm run deploy
```

## 📊 Данные

Приложение использует JSON файлы с данными о:
- **Специях** (`/public/data/spices.json`) - 43 специи с химическими соединениями и вкусовыми профилями
- **Продуктах** (`/public/data/products.json`) - 25 категорий продуктов с характеристиками
- **Методах приготовления** (`/public/data/cooking-methods.json`) - 16 методов с температурными режимами

## 🧪 Алгоритм анализа

Система анализа учитывает:
1. **Совместимость с продуктом** - Соответствие специй выбранному продукту
2. **Совместимость с методом** - Подходящие температурные режимы
3. **Гармония специй** - Взаимодействие между выбранными специями
4. **Химическая совместимость** - Взаимодействие химических соединений

## 📱 PWA функции

- **Офлайн работа** - Кэширование данных для работы без интернета
- **Установка на устройство** - Возможность установки как нативное приложение
- **Push уведомления** - Уведомления о новых функциях
- **Адаптивный дизайн** - Оптимизация для всех размеров экранов

## 🤖 Telegram WebApp

- **Haptic Feedback** - Тактильная обратная связь
- **Интеграция с UI** - Использование нативных элементов Telegram
- **Пользовательские данные** - Доступ к информации о пользователе
- **Кнопки управления** - MainButton и BackButton

## 🚀 Деплой

Проект автоматически деплоится на GitHub Pages при пуше в ветку `main`.

### Настройка GitHub Pages
1. Перейдите в Settings → Pages
2. Выберите Source: GitHub Actions
3. Workflow автоматически запустится при следующем коммите
